Test de Seguridad Informática

1. ¿Cuál de los siguientes NO es uno de los cuatro pilares fundamentales de la seguridad informática?



Respuesta correcta: d) Rentabilidad
Los cuatro pilares son: Confidencialidad, Integridad, Disponibilidad y Fiabilidad.
2. ¿Qué herramienta se utiliza para realizar escaneos de red y detección de servicios activos?



Respuesta correcta: b) Nmap
Nmap (Network Mapper) es la herramienta especializada en exploración de redes y detección de servicios.
3. ¿Qué tipo de criptografía utiliza una única clave para cifrar y descifrar?



Respuesta correcta: c) Simétrica
La criptografía simétrica utiliza la misma clave para cifrar y descifrar, mientras que la asimétrica usa un par de claves.
4. ¿Qué comando se usa en Linux para generar un par de claves SSH?



Respuesta correcta: a) ssh-keygen
El comando ssh-keygen es el estándar para generar pares de claves SSH pública y privada.
5. ¿Qué porcentaje de brechas de datos se debe a errores humanos según el documento?



Respuesta correcta: c) 95%
Según los documentos, el 95% de las brechas de datos exitosas se deben a errores humanos.
6. ¿Qué herramienta forense es de código abierto y proporciona una interfaz gráfica?



Respuesta correcta: c) Autopsy
Autopsy (The Sleuth Kit) es la plataforma forense open source con interfaz gráfica.
7. ¿Qué protocolo es considerado inseguro por transmitir credenciales en texto plano?



Respuesta correcta: c) Telnet
Telnet transmite todas las comunicaciones, incluidas las credenciales, en texto plano sin cifrar.
8. ¿Qué comando en Linux se usa para ver las ACLs de un archivo?



Respuesta correcta: c) getfacl
getfacl muestra las Access Control Lists de archivos y directorios.
9. ¿Qué tipo de ataque cifra archivos y exige un rescate?



Respuesta correcta: b) Ransomware
El ransomware cifra los archivos del sistema y exige un rescate para la clave de descifrado.
10. ¿Qué algoritmo de hash se considera obsoleto y vulnerable?



Respuesta correcta: c) MD5
MD5 y SHA-1 son considerados obsoletos y vulnerables a colisiones.
11. ¿Qué opción de Nmap se usa para detectar el sistema operativo del objetivo?



Respuesta correcta: b) -O
La opción -O (detección OS) permite identificar el sistema operativo del objetivo.
12. ¿Qué principio de seguridad asegura que los datos no sean alterados?



Respuesta correcta: b) Integridad
La integridad garantiza que los datos permanezcan íntegros y no sean modificados de manera no autorizada.
13. ¿Qué herramienta se usa para análisis estático de código?



Respuesta correcta: b) SonarQube
SonarQube es una herramienta especializada en análisis estático de código.
14. ¿Qué comando se usa en Linux para eliminar todas las ACLs de un archivo?



Respuesta correcta: b) setfacl -b
setfacl -b elimina todas las Access Control Lists de un archivo o directorio.
15. ¿Qué tipo de autenticación utiliza características físicas como huellas dactilares?



Respuesta correcta: b) Biométrica
La biometría utiliza características físicas o conductuales únicas para la autenticación.
16. ¿Qué estrategia de backup recomienda tener al menos 3 copias en 2 medios diferentes?



Respuesta correcta: c) Estrategia 3-2-1
La estrategia 3-2-1 establece: 3 copias totales, en 2 medios diferentes, con 1 copia fuera del sitio.
17. ¿Qué herramienta de Nmap permite usar scripts para detectar vulnerabilidades?



Respuesta correcta: c) --script
La opción --script permite ejecutar scripts NSE (Nmap Scripting Engine) para diversas funciones incluyendo detección de vulnerabilidades.
18. ¿Qué capa de red es vulnerable a ataques como ARP Poisoning?



Respuesta correcta: b) Enlace
ARP Poisoning ataca la capa de enlace (capa 2 del modelo OSI) mediante envenenamiento de tablas ARP.
19. ¿Qué comando en Linux reinicia el servicio SSH?



Respuesta correcta: c) sudo systemctl restart sshd
En sistemas con systemd, el servicio SSH se llama 'sshd', no 'ssh'.
20. ¿Qué tipo de criptografía utiliza claves pública y privada?



Respuesta correcta: b) Asimétrica
La criptografía asimétrica utiliza un par de claves: pública (para cifrar) y privada (para descifrar).
21. ¿Qué herramienta se usa para interceptar y analizar tráfico de red?



Respuesta correcta: b) Wireshark
Wireshark es el analizador de protocolos de red más utilizado para capturar y analizar tráfico.
22. ¿Qué significa RTO en recuperación ante desastres?



Respuesta correcta: a) Recovery Time Objective
RTO (Recovery Time Objective) es el tiempo máximo aceptable de inactividad después de un incidente.
23. ¿Qué comando en Windows se usa para generar claves SSH con interfaz gráfica?



Respuesta correcta: a) PuttyGen
PuttyGen es la herramienta gráfica incluida en PuTTY para generar pares de claves SSH en Windows.
24. ¿Qué tipo de amenaza implica el acceso físico no autorizado a un servidor?



Respuesta correcta: b) Física
Las amenazas físicas involucran el acceso directo o daño a los componentes físicos del sistema.
25. ¿Qué opción de Nmap realiza un escaneo UDP?



Respuesta correcta: c) -sU
La opción -sU realiza escaneos UDP para detectar servicios que utilizan este protocolo.
26. ¿Qué comando se usa para copiar la clave pública a un servidor remoto?



Respuesta correcta: b) ssh-copy-id
ssh-copy-id es el comando específico para copiar automáticamente la clave pública al servidor remoto.
27. ¿Qué permisos se recomiendan para el archivo authorized_keys en el servidor?



Respuesta correcta: c) 600
chmod 600 establece permisos de lectura/escritura solo para el propietario, garantizando seguridad.
28. ¿Qué algoritmo de cifrado asimétrico está basado en curvas elípticas?



Respuesta correcta: c) ECC
ECC (Elliptic Curve Cryptography) ofrece mayor seguridad con claves más cortas que RSA.
29. ¿Qué herramienta se menciona para análisis de inyecciones SQL?



Respuesta correcta: a) SQLMap
SQLMap es la herramienta especializada en detección y explotación de vulnerabilidades de inyección SQL.
30. ¿Qué representa el principio de "mínimo privilegio"?



Respuesta correcta: b) Otorgar solo los permisos necesarios
El principio de mínimo privilegio establece que usuarios y procesos deben tener solo los permisos esenciales para realizar sus tareas.
31. ¿Qué tipo de backup copia solo los archivos modificados desde el último backup de cualquier tipo?



Respuesta correcta: c) Incremental
El backup incremental es el más rápido pero requiere todos los incrementos para una restauración completa.
32. ¿Qué ataque consiste en suplantar la identidad en comunicaciones de red?



Respuesta correcta: b) Spoofing
El spoofing implica falsificar direcciones IP, MAC o otros identificadores para suplantar identidad.
33. ¿Qué herramienta de Nmap realiza un escaneo SYN?



Respuesta correcta: a) -sS
El escaneo SYN (-sS) es el método por defecto de Nmap, más rápido y sigiloso que el connect().
34. ¿Qué función criptográfica genera una huella digital única de los datos?



Respuesta correcta: c) Función hash
Las funciones hash como SHA-256 generan un valor único fixed-length que representa los datos originales.
35. ¿Qué comando setfacl asigna permisos por defecto en un directorio?



Respuesta correcta: b) setfacl -d -m
La opción -d (default) establece ACLs por defecto que se heredarán en nuevos archivos del directorio.
36. ¿Qué tipo de datos es vulnerable a ataques de "man-in-the-middle"?



Respuesta correcta: b) Datos en tránsito
Los datos en tránsito son susceptibles a interceptación y manipulación durante su transmisión por la red.
37. ¿Qué herramienta se menciona para auditoría de redes wireless?



Respuesta correcta: a) Aircrack-ng
Aircrack-ng es la suite especializada en auditoría de seguridad de redes inalámbricas.
38. ¿Qué característica tiene la criptografía asimétrica frente a la simétrica?



Respuesta correcta: c) No requiere intercambio previo de claves secretas
La criptografía asimétrica resuelve el problema de distribución de claves mediante el uso de pares público/privado.
39. ¿Qué porcentaje de vulnerabilidades de software se deben a errores de programación evitables?



Respuesta correcta: c) 90%
Según los documentos, el 90% de las vulnerabilidades de software son debidas a errores comunes y evitables.
40. ¿Qué comando de Nmap realiza un "ping sweep" para descubrir hosts activos?



Respuesta correcta: b) nmap -sn
La opción -sn (ping scan) descubre hosts activos sin escanear puertos.
41. ¿Qué tipo de ACL controla eventos de auditoría en el sistema?



Respuesta correcta: b) SACL
SACL (System Access Control List) define qué eventos de acceso deben ser auditados.
42. ¿Qué temperatura óptima se recomienda para salas de servidores?



Respuesta correcta: b) 18-24°C
Mantener la temperatura entre 18-24°C optimiza el rendimiento y prolonga la vida útil del hardware.
43. ¿Qué herramienta de análisis forense se especializa en memoria RAM?



Respuesta correcta: c) Volatility Framework
Volatility Framework es la herramienta especializada en análisis forense de memoria RAM.
44. ¿Qué ataque utiliza dispositivos IoT comprometidos para generar tráfico DDoS?



Respuesta correcta: b) Mirai
El botnet Mirai utilizó 600,000 dispositivos IoT comprometidos para ataques DDoS masivos.
45. ¿Qué significa RPO en planes de recuperación?



Respuesta correcta: a) Recovery Point Objective
RPO define la cantidad máxima de datos que se pueden perder desde el último backup válido.
